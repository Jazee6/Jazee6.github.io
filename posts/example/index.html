Python 示例

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 生成斐波那契数列
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")

这个示例文档展示了 Markdown 的主要语法功能，包括：
